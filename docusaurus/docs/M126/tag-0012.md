**SSH Konfigurieren**

`conf t` -> in config mode gehen  
`username [USERNAME] password [PASSWORD]` -> user mit passwort für ssh erstellen  
`aaa authentication ssh console LOCAL` -> user für ssh erlauben  
`crypto key generate rsa modulus 1024` -> rsa keypair erstellen (überschreiben)  
`ssh [CLIENT-IP] [CLIENT-NETMASK] [VLAN]` -> ssh für bestimmte ip erlauben (VLAN = inside)  
Per SSH auf Cisco ASA verbinden
