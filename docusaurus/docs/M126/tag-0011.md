Per Putty über COM-Port auf Cisco ASA verbinden
![Putty](/img/m126/cisco_asa_com_connection.png)

**Reset**

Gerät von Strom nehmen, Putty offen lassen

Wieder einstecken und während Boot ESCAPE in Putty drücken

`confreg 0x41` -> Update config register  
`boot` -> booten  
`en` -> um in priviledged mode zu gelangen, passwort setzen  
`write erase` -> startup config löschen  
`conf t` -> in config mode zu gelangen  
`config-register 0x01` -> configuration register resetten  
`exit` -> config mode verlassen  
`show version` -> version anzeigen, sollte `Configuration register is 0x41 (will be 0x1 at next reload)` sein  
`wr` -> neuen settings speichern (write memory)  
`reload` -> gerät rebooten  
`en`  
`conf t`  
`config factory [IP]` -> ip setzen, default 192.168.1.1/24  
`wr`  
`sh start` -> start config anschauen, ip überprüfen  
![Startup Config](/img/m126/cisco_asa_startup_config.png)  
`sh run` -> run config anschauen, ist die laufende config (im RAM)  
`sh interface ip brief` -> VLAN config anschauen
