# Tag 11

:::note
OpenSSL installieren -> sudo apt install openssl

mod_rewrite aktivieren -> sudo a2enmod rewrite

Cert generieren -> sudo openssl req -x509 -nodes -days 365 -newkey rsa:2048 -keyout /etc/ssl/private/apache-selfsigned.key -out /etc/ssl/certs/apache.crt

Config überprüfen -> /etc/apache2/ports.conf
```config
<IfModule mod_ssl.c>
    Listen 443
</IfModule>
```

SSL mod aktivieren -> sudo a2enmod ssl

Apache reloaden -> sudo systemctl reload apache2

Website config (/etc/apache2/sites-available/ssl.conf) | Kann auch in die gleiche Config geschrieben werden
```config
<VirtualHost *:443>
    SSLEngine on
    SSLCertificateFile /etc/ssl/certs/apache.crt
    SSLCertificateKeyFile /etc/ssl/private/apache.key

    # Pfad zu den Webinhalten
    DocumentRoot /var/www/test
</VirtualHost>
```

Website aktivieren -> sudo a2ensite ssl.conf
