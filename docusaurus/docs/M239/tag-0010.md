500 Fehler -> falsche Konfiguration, Überlastung, wie viele Bytes werden versendet?  
hohe Verzögerung -> falsche Konfiguration  
Site not Responding -> Betriebszeit, lange kein Reboot?, kleiner Ausfall

**Installation mod_info mod_status**
sudo a2enmod info # modul mod_info freischalten (mod_status ist schon freigeschaltet)

config für mod_status (in 000-default.conf)  
```
<Location /server-status>
        SetHandler server-status
        Order deny,allow
        Deny from all
        Allow from 127.0.0.1
</Location>
```

config  für mod_info (in 000-default.conf)  
```
<Location /server-info>
    SetHandler server-info
    Order deny,allow
    Deny from all
    Allow from 127.0.0.1
</Location>
```
:::
